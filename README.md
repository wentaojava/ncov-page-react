
## ncov-react-page

react框架实现简单的页面，用来展示2019新型冠状病毒的疫情数据

### `展示数据包括`

当日累积确诊人数，死亡人数，治愈人数<br />
疫情地图展示<br/>
根据省份展示所属省份下各城市疫情数据
